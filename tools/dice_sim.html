Glückritter Inc — Dice Pool Simulator

D10 pool resolution system

7, 8, 9 → hit 1, 2 → failure 0 → crit indicator 3–6 → neutral
Net = hits − failures  ·  Critical success when 0s > failures  ·  Critical failure when 0s > hits
Net → outcome: ≤−3 catastrophic failure  ·  −2/−1 failure  ·  0 partial/costly  ·  +1/+2 success  ·  +3/+4 strong  ·  +5+ overwhelming
A critical flag amplifies the outcome — the game decides what that means per context.
